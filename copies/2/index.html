CSS Transitions & Animations

CSS Transitions

Transitions allow you to change property values smoothly over a given duration.

Basic Transition Example

Hover over the container to see the transition effect:


/* Transition Properties */
.transition-box {
    width: 100px;
    height: 100px;
    background: #3498db;
    transition: all 0.3s ease;
    /* property name | duration | timing function | delay */
}

.transition-demo:hover .transition-box {
    transform: translateX(100px);
    background: #e74c3c;
    border-radius: 50%;
}

/* Transition Shorthand */
.element {
    transition: property duration timing-function delay;
}

/* Multiple Transitions */
.element {
    transition: 
        background-color 0.3s ease,
        transform 0.5s ease-in-out 0.2s;
}

/* Transition Timing Functions */
.ease { transition-timing-function: ease; } /* Default */
.linear { transition-timing-function: linear; }
.ease-in { transition-timing-function: ease-in; }
.ease-out { transition-timing-function: ease-out; }
.ease-in-out { transition-timing-function: ease-in-out; }
.cubic-bezier { transition-timing-function: cubic-bezier(0.1, 0.7, 1.0, 0.1); }

/* Keyframes Animation */
@keyframes example {
    from { background-color: red; }
    to { background-color: yellow; }
}

/* Using the animation */
.element {
    animation-name: example;
    animation-duration: 4s;
    animation-timing-function: ease;
    animation-delay: 1s;
    animation-iteration-count: infinite;
    animation-direction: alternate;
}

/* Animation Shorthand */
.element {
    animation: example 4s ease 1s infinite alternate;
}

/* Multiple Keyframes */
@keyframes complexExample {
    0% {
        transform: translate(0, 0);
        opacity: 0;
    }
    50% {
        transform: translate(100px, 50px);
        opacity: 1;
    }
    100% {
        transform: translate(200px, 0);
        opacity: 0;
    }
}

/* Bounce Animation Example */
@keyframes bounce {
    0%, 100% {
        transform: translateY(0);
    }
    50% {
        transform: translateY(-30px);
    }
}

.bounce {
    animation: bounce 2s infinite;
}

Animation Properties

  • animation-name: Specifies the name of the @keyframes animation
  • animation-duration: How long an animation should take to complete
  • animation-timing-function: Speed curve of the animation
  • animation-delay: Delay before the animation starts
  • animation-iteration-count: How many times an animation should be played
  • animation-direction: Whether an animation should play in reverse on alternate cycles
  • animation-fill-mode: Specifies styles for element when animation is not playing
  • animation-play-state: Specifies whether the animation is running or paused

Animation Performance Tips

Use Transform and Opacity

For best performance, animate properties that don't trigger layout changes or repaints:

  • transform: translate, scale, rotate, skew
  • opacity: Fading elements in and out

Avoid Animating These Properties

These properties are expensive to animate as they trigger layout recalculations:

  • width, height
  • margin, padding
  • top, right, bottom, left
  • border-width
  • font-size

/* Good for performance */
.element {
    transform: translateX(100px);
    opacity: 0.5;
}

/* Bad for performance */
.element {
    margin-left: 100px;
    width: 200px;
}

Use will-change Property

The will-change property hints to browsers how an element is expected to change:


.element {
    will-change: transform, opacity;
}

Note: Use will-change sparingly and only on elements that definitely will change.
